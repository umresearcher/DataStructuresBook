    Chapter 2 Memory Allocation in C++

    C++ provides fine-grained control over memory through several types of variables. To understand how memory is allocated and accessed, we categorize variables as follows:
    • Ordinary (value) variables: These directly hold values and are typically allocated on the stack (for both built-in and user-defined types). Examples include:
